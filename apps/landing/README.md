## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

## Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```
